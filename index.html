About Me

Hello! My name is Jessica, and I am a math enthusiast turned Java developer. I have always been in the business of problem solving. I majored in math at the University of Georgia and then taught high school Geometry and Algebra II. Nothing could have prepared me, however, for how much my love for problem solving would grow when I dove into the world of software development. My Projects

Teacher's Pet

A web app that allows teachers to grade tests on a curve by providing built in algorithms as well as grade distribution graphs to help teachers decide which curve is appropriate without having to know the math behind it. It also provides quick and easy parent contact functionality by populating email templates with individual student grade data and can email entire sets of students at once based on different criteria. This application was built using Java with Spring connecting to a PostgreSQL server through Hibernate on the back end and using AngularJS on the front end.

Bank System

A command line application that allows a user to enter information about his/her bank accounts and then perform transactions on those accounts. This project makes use of background threads to calculate interest on savings and retirement accounts, writes data to file to store account information, and showcases principles of inheritance and polymorphism in working with different types of bank accounts.

Chocolate Factory

A simple web application that allows a user to enter amounts of small (1 kg) and large (5 kg) bars of chocolate into his/her inventory and uses an algorithm to determine if it is possible to make the desired amount of “goal” kilograms of chocolate from that inventory. This project showcases thorough unit testing and also implements javascript animation.
